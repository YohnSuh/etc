HTML이란?

- 웹 페이지를 만드는 뼈대로 태그를 사용하여 작성합니다.
  태그는 <여는태그></닫는태그>로 작성하며,
  일부 태그는 '닫는태그' 없이 사용이 가능합니다.
  [ 주의! 닫는 태그가 없으면 오류가 발생할 수 있습니다. ]

  ex. <div> 내용입니다. </div>

- 태그에는 고유한 이름을 부여할 수 있습니다.
  여는 태그 안에 'class'속성이나 'id'속성을 넣어 이름을 작성하면 됩니다.
  이름은 영문 소문자를 사용하고 띄어쓰기 대신 '-', '_'을 사용하여 단어를 연결합니다.
  이름의 중간과 끝에 숫자를 넣을 수 있지만 첫 문자로 숫자나 특수기호를 사용할 수 없습니다.
  작성 방법은 태그 안에 'class="이름"' 또는 'id="이름"'으로 작성합니다.
  여러 태그에 같은 class명을 중복으로 사용할 수 있지만 id명은 중복 사용이 불가능합니다.
  [ 주의! 태그명과 클래스/아이디 속성 사이에는 꼭 띄어쓰기를 해주세요. ]

  ex. <div class="name"></div>
      <div class="name1"></div>
      <div class="class-name"></div>
      <div id="name"></div>
      <div id="name2"></div>
      <div id="id_name"></div>

- 태그 안에 태그를 넣어 사용할 수 있습니다. 태그 안의 태그를 작성 시 'Tab'키보드를 사용하여
  들여쓰기를 하면 가독성을 높일 수 있습니다.

  ex.
  <div>
     <div>
         내용입니다.
     </div>
  </div>

CSS란?

- html로 작성한 웹 문서를 꾸며주는 것으로 주로 <style></style>태그 안에
  코드를 작성하는 방법을 사용합니다.

  [ css 문법 ]
  <style>
      선택자 {
            속성: 속성값;
      }
  <style>
  => 선택자는 스타일을 입히고자 하는 요소(태그)를 말합니다.
  => 속성은 어떤 스타일을 입힐지 종류를 말하고 속성값은 해당 속성이 가지는 스타일값을 말합니다.
  => 선택자에는 태그 외에도 태그의 이름을 넣어 작성할 수 있습니다.
        class명의 경우 이름 앞에 '.'을 붙여 작성하며, id명은 이름 앞에 '#'를 붙여 작성합니다.
  
  ex.
  <style>
      div {background: red;}
      => div태그의 배경(속성)을 'red(속성값)' 색으로 바꾸겠다.
      .class-name {color: blue;}
      => class명이 'class-name'인 태그의 글자색(속성)을 'blue(속성값)' 색으로 바꾸겠다.
      #id-name {font-weight: bold;}
      => id명이 'id-name'인 태그의 글자굵기(속성)을 'bold(속성값)'로 바꾸겠다.
  <style>

- 'style'태그를 사용하는 방법 외에도 태그 안에 'style'속성을 넣어 css를 적용할 수도 있습니다.
  이 때, 태그 안에 'style'속성을 넣어 적용한 css가 'style'태그를 사용하여 적용한 css보다 우선적으로 적용됩니다.
  
  [ css 문법 ]
  <여는태그 style="속성: 속성값;"><닫는태그>

  ex. <div style="color": red;"><div>

- '속성값'과 ';' 사이에 '!important'를 작성할 시 해당 스타일이 가장 우선적으로 적용됩니다.
  [ 주의! '!important'앞에 꼭 띄어쓰기를 해주세요. ]

  ex. <div style="color": red !important;"><div>

다음카페 꾸미기

1) 폰트 적용하기

* 주의! 눈누폰트 사용 기준 입니다.
(1) 눈누폰트 페이지에 들어간다.
(2) 원하는 폰트를 선택한 후 우측의 '웹폰트 사용' 코드를 복사한다.
    - @font-family {~~} 라고 적힌 코드입니다.
(3) '스타일태그' 안에 복사한 코드를 붙여넣기 한다.
(4) '{}'안에 'font-family: ~~ ;'라고 적힌 줄을 복사한다.
(5) 'style'태그 안에 선택자와 '{}'를 작성하고 '{}'안에 복사한 'font-family: ~;'를 붙여넣는다.

ex.
<style>
    @font-face {
        font-family: 'Pretendard-Regular';
        src: url('https://cdn.jsdelivr.net/gh/Project-Noonnu/noonfonts_2107@1.1/Pretendard-Regular.woff') format('woff');
        font-weight: 400;
        font-style: normal;
    }
    * {font-family: 'Pretendard-Regular';}
    => 선택자에 '*'를 사용하면 모든 태그를 선택할 수 있습니다.
</style>
